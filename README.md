# Smart Task Tracker System

## Project Description
Smart Task Tracker System is a Java-based project developed to help users manage and track daily tasks efficiently. It allows users to create tasks, assign priorities, monitor progress, and update task status in an organized manner.

## Modules
1. User Authentication Module
2. Task Creation Module
3. Task Assignment Module
4. Task Status Tracking Module
5. Deadline Reminder Module
6. Task Report Generation Module

## Technologies Used
- Java
- MySQL
- JDBC
- Swing / AWT

## Features
- Add new tasks
- Update task details
- Track task progress
- Set task deadlines
- Generate task reports

## Future Scope
This project can be enhanced by adding notifications, cloud storage, team collaboration, and mobile support in future versions.
